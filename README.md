# Playnite Local Metadata Importer
The addon allows for importing media like icons, backgrounds, covers, logos and videos from locally available files on your disks based on romname or game name. 
The addon can aid in converting or importing from existing, other emulation frontends which base the media filename on the rom name. 
It can also be used in conjunctions with skraper.net or skyscraper media scraping tools to import their downloaded data.

## Configuration
Configuration of the addon is simple, in the addon settings click "add new page" to add a new page to the list then select the platform and their media folders. Do this for each platform you wish to import data for.

## importing
Once configuration has been done you can use the extensions menu to access the import menus of Local Metadata Importer, you can choose to import all media or individual media for all games, filtered games or selected games.
The import process will show a progress bar.
